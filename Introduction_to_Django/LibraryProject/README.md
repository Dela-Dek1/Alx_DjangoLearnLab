# LibraryProject

## 📌 Project Description
LibraryProject is a Django-based web application for managing a library system. It allows users to browse, borrow, and return books while maintaining records of transactions.

## 🚀 Features
- User authentication and management
- Book catalog and search functionality
- Borrowing and return system
- Admin panel for managing library resources

## 🛠️ Technologies Used
- Python 3.13
- Django 5.x
- SQLite (default database)
- Bootstrap (for frontend styling)
